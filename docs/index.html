How it works

Simple pipeline, great results

From PDF to CBZ in five steps — fully automated.

1

Detect

Scans the first 50 pages — if ≥30% are tall, it's manhwa.

2

Extract

Pulls high-quality images from each PDF page using PyMuPDF.

3

Process

Applies gamma, grayscale, autocontrast, and device resize.

4

Split

Manhwa pages are divided into perfectly-sized screen sections.

5

Package

All images are zipped into a .cbz file with JPEG compression.
